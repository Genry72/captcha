Как это работает?

Наш сервис предоставляет HTTP API для распознавания текста на изображениях, идеально подходящее для простых капч. Вы отправляете изображение, а в ответ получаете распознанный текст и его координаты.

Быстро

0.2 сек на капчу — до 5 в секунду

Надежно

Высокая точность распознавания

Легкая интеграция

Использование API

1

Получите доступ к API

Свяжитесь с нашим Telegram-ботом для получения API ключа и полного URL для запросов

Получить доступ
2

Пример запроса

Отправьте POST запрос на полный URL (полученный от бота) с вашим изображением и API токеном

bash 
curl -X POST 'YOUR_FULL_URL' \
  -H 'vstoken: YOUR_API_TOKEN' \
  -F 'file=@/path/to/your/captcha.png'
3

Пример успешного ответа

В случае успешного распознавания, вы получите JSON-ответ со статусом 200

json 
[
  {
    "bbox": [
      [34, 16],
      [143, 16],
      [143, 52],
      [34, 52]
    ],
    "text": "kw9rN",
    "conf": 0.9985
  }
]

Описание полей ответа

bbox

Массив с координатами, определяющими рамку вокруг распознанного текста

text

Распознанный текст

conf

Уверенность модели в правильности распознавания (0-1)
